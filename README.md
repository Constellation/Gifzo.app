## Gifzo.app
Gifzo Mac Client

http://gifzo.net/

## Build

### CocoaPods

1. Install CocoaPods at first time

```
$ [sudo] gem install cocoapods
$ pod setup
```

2. Install pods for resolve dependencies

```
$ pod install
```

3. Open `.xcworkspace` file

```
$ open Gifzo.xcworkspace
```

4. Enjoy!
